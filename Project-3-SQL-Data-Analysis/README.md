# Automobile Data Analysis Using SQL

## Project Overview

This project was completed as part of the DecodeLabs Data Analytics Internship (Project 3: SQL Data Analysis).

The objective of this project was to use SQL queries to analyze an automobile dataset and extract meaningful business insights. The analysis focuses on vehicle pricing, fuel types, body styles, horsepower, and manufacturer trends.

## Key Business Insights

1. Jaguar has the highest average vehicle price among all manufacturers in the dataset.

2. Mercedes-Benz, Porsche, and BMW also rank among the most expensive automobile brands.

3. Gas-powered vehicles dominate the dataset, indicating higher market representation.

4. Dodge has the highest number of vehicles in the dataset, followed by BMW and Mercedes-Benz.

5. Luxury brands generally command significantly higher prices than mainstream automobile manufacturers.

6. Higher horsepower and larger engine sizes are generally associated with higher vehicle prices.

7. Certain body styles are more common than others, indicating customer preference trends.
